# Module tile icons — attribution

The glyphs in these tiles are recoloured (white) and composited onto the Nexum Air
teal→cyan brand squircle. Two upstream icon sets are used:

- **Material Symbols** (Google) — **Apache License 2.0**, see `LICENSE`. Source
  package `@material-symbols/svg-400` (rounded, filled). Used for all tiles except
  Projects.
- **Phosphor Icons** — **MIT License**, see `LICENSE-phosphor`. Source package
  `@phosphor-icons/core` (fill weight). Used for the Projects tile (`hard-hat`),
  since Material Symbols has no standalone hard-hat glyph.

No attribution is required in the application UI under either licence; these
notices and the bundled licence files satisfy the retention requirements.
